# 🌱 My Digital Garden

Welcome to my **Digital Garden** — a growing collection of thoughts, notes, ideas, learning logs, and resources I’m cultivating over time.

This project contains:
- 🍃 Notes on various topics
- 🔗 Interlinked content
- 🧠 Personal reflections and evolving ideas
- 🛠 Built using Markdown and static site tooling

---

## 📌 Description

This repository is my personal digital garden — a *living*, evolving space of knowledge that I update regularly. Unlike a traditional blog, this garden showcases my thinking process, evergreen notes, and learnings from day-to-day exploration.

Ideas are treated like plants — they start small, grow over time, and sometimes transform into something deeper.
